# Retail Sales SQL Analysis Project

## 📌 Project Overview
This project demonstrates SQL-based data analysis using a retail sales dataset. The objective is to explore sales performance, analyze customer purchasing behavior, and generate business insights using PostgreSQL.

---

## 🎯 Objectives
- Analyze retail sales data.
- Identify sales trends and patterns.
- Calculate revenue and order statistics.
- Perform category-wise sales analysis.
- Answer business-related questions using SQL.

---

## 🛠️ Tools & Technologies
- PostgreSQL
- pgAdmin 4
- SQL
- GitHub

---

## 📂 Dataset
The dataset contains retail sales transaction records with information such as:
- Transaction ID
- Sale Date
- Customer ID
- Category
- Product
- Quantity
- Price per Unit
- Total Sale

---

## 📊 SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- LIMIT
- Aggregate Functions (SUM, COUNT, AVG, MAX, MIN)
- Date Functions
- Aliases

---

## 📈 Business Questions Solved
- Total number of sales transactions
- Total revenue generated
- Monthly sales trend
- Category-wise sales analysis
- Top-selling categories
- Average sales per category
- Customer purchase analysis
- Highest sales transaction
- Sales by date
- Product performance analysis
